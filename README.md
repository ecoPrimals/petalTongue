## 🚀 Quick Start

### Run petalTongue
```bash
cargo run --release
```

### With biomeOS Integration
```bash
# Start biomeOS first
cargo run --release

# petalTongue will auto-discover via mDNS
```

### Run Tests
```bash
cargo test --all-features
```
