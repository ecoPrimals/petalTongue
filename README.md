# 🌸 petalTongue

**Universal UI Platform for ecoPrimals**

A sensory coordination layer that adapts to any device and embeds any application.

---

## 🎯 Quick Start

```bash
# Run default UI
cargo run --release --bin petal-tongue

# Run with Doom MVP
cargo run --release --bin petal-tongue -- --scenario sandbox/scenarios/doom-mvp.json

# Run tests
cargo test

# Generate coverage report
cargo llvm-cov --html
```

## ✨ Features

### **Universal Panel System**
Embed any application as a panel:
- 🎮 Games (Doom MVP implemented)
- 🌐 Web browsers (planned)
- 🎬 Video players (planned)
- 💻 Terminals (planned)
- 🎨 Custom tools

### **Sensory Capability System**
Discovers device I/O instead of hardcoding:
- **Outputs**: Visual (2D/3D), Audio, Haptic
- **Inputs**: Pointer, Keyboard, Touch, Gesture
- Adapts to: Desktop, phone, watch, VR, neural interfaces

### **Interactive Canvas**
- Create nodes (double-click)
- Connect nodes (drag)
- Delete nodes (Delete key)
- Capability validation
