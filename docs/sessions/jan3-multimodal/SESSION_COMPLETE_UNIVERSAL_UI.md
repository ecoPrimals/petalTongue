## 🔊 How It Works

**Every 200ms**:
1. Dashboard reads CPU (75%) and Memory (60%)
2. CPU → 1550Hz sine wave (left channel)
3. Memory → 400Hz @ 60% volume (center)
4. Mix polyphonically → WAV file → Play via system player
5. User hears real-time audio representing system state!

**User Control**: Press 'A' → Enable Audio Sonification → Adjust Volume
